# Simple ETH Faucet DApp

A clean and beginner-friendly Web3 project demonstrating how to:

- Write a Solidity smart contract
- Deploy it
- Connect a frontend with Web3
- Send ETH from a faucet contract

This repo is intentionally **flat structured** so beginners can easily understand each file.

---

## Features

• Request test ETH from faucet  
• Owner can deposit funds  
• Cooldown timer between claims  
• MetaMask connection  
• Minimal frontend UI  

---

## Tech Stack

- Solidity
- Ethers.js
- HTML + Vanilla JS
- Hardhat
